How to build
----------------
```bash
./build.sh
```
Remeber to change the `INTEL_ROOT` to your custom path.

How to run
-------------
```bash
./run_bench.sh
```
Remember to change the `KMP_AFFINITY` to fit your machine.
You may also try other settings like `MKL_DYNAMIC`, `OMP_DYNAMIC` and `MKL_NUM_THREADS`.
